# Weather app

A simple weather site where you can view the weather of your city

## Features

The site has got two page:
- One to see the weather
- One with all the details about the project for programmers

## Technologies

Built with:
```
- C#
- HTML
- CSS
- JS
- ASP.NET Core
- JSON
- XML
```
